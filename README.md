# C++ Snake

Basic C++ made from scratch by me (@jvillemare).

## Run

To run the game, simply execute:

- Mac: `make run_mac`
- Linux: `make run_linux`
- Windows: `make run_win`

## Cleanup

To cleanup, simply execute:

```sh
make clean
```
